## Installation

<Installation>
<div>
```bash title="Terminal"
npx shadcn@latest add https://9ui.dev/r/sonner.json
```
</div>

<Steps>

<Step>Copy and paste the following code into your project.</Step>

<Source name="sonner" />

<Step>Add Toaster to your app layout.</Step>

```tsx title="app/layout.tsx"
import { Toaster } from "@9ui/uisonner";

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <head />
      <body>
        <main>{children}</main>
        <Toaster />
      </body>
    </html>
  );
}
```

</Steps>
</Installation>

## Usage

```tsx title="Imports"
import { toast } from "sonner";
```

```tsx title="Toast"
toast("Toast is working!");
```
